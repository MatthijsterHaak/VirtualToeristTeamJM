LEES DIT VOOR INSTALLATIE EN ZELF RUNNEN VAN DE APP

1. Installeer NodeJS
2. Clone Repository door op clone te drukken in de Azure Devops omgeving
3. Open de folder in je eigen editor
4. Type 'npm install -g expo-cli' in de terminal
5. Type 'npm install' in de terminal
6. Type 'expo start' in de terminal
7. Open de Expo app op de telefoon en scan de QR-Code
