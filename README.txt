Installatie expo omgeving 

1. Installeer NodeJS als deze nog niet aanwezig is op uw systeem; (https://nodejs.org/en/download//)
2. Type 'npm install -g expo-cli' in de terminal en run de commando;
3. Open de folder met de code in de door uw gekozen editor;
4. Type 'npm install' in de terminal en run het commando;
5. Download de Expo app op uw telefoon of uw telefoon emulator zoals Android Studio;
6. Type 'expo start' in de terminal en run het commando, er zal nu een venster worden geopend in uw browser;
7. Open de Expo app op de telefoon en scan de QR-Code die wordt weergegeven in het venster of open het via uw emulator door aan de linker kan van het venster op 'Run on Android device/emulator' of 'Run on iOS simulator' te drukken.

voor meer inforamtie van de installatie van react native: 
https://reactnative.dev/docs/environment-setup
